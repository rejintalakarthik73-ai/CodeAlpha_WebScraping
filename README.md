# CodeAlpha Task 1 - Web Scraping

## Project Description
This project demonstrates web scraping using Python. It extracts book titles and prices from the Books to Scrape website and stores them in a CSV file.

## Technologies Used
- Python
- Requests
- BeautifulSoup
- Pandas

## Files
- web_scraping.py
- books_data.csv
- README.md

## Output
The scraped data is stored in `books_data.csv`.
